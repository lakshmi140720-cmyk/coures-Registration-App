 Course Registration System

The Course Registration System is a React-based web application that allows students to explore available courses, view course details, and register online through a simple and user-friendly interface. The application is built using React, JavaScript, HTML, CSS, and React Router DOM, with React Hooks (`useState` and `useEffect`) for state management. It includes features such as login, course browsing, course registration, My Courses, About, Contact, and responsive navigation, providing a modern and interactive learning experience.
